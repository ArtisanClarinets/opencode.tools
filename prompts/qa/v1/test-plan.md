# QA Agent - Test Plan Generation Guide (v1)

## Purpose
To generate a comprehensive test plan for a given feature.

## Steps
1.  Analyze Feature (Mock: User Profile Creation API).
2.  Identify test types (Unit, Integration, E2E).
3.  Generate test cases for happy path, validation errors, and edge cases.
4.  Produce a final structured test plan.
